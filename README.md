# MCItemList
打开我的世界投影模组导出的材料列表并以非常直白的方式查看，还支持多人协同处理，方便了服务器的备货工作。

# 快速开始
## Windows

* 从[Releases](https://github.com/Akttoer/MCItemList/releases)下载最新版本
* 解压压缩包
* 双击打开main.exe

## MacOS/Linux

* 从[Releases](https://github.com/Akttoer/MCItemList/releases)下载最新版本
* 安装python并从requirements.txt安装软件包
* python main.py
